ACLIoT –Climatização e Iluminação com IoT

Explorando Soluções Inteligentes para Eficiência Energética Atualmente em fase de desenvolvimento, o projeto ACLIoT integra tecnologias de IoT para monitoramento básico de climatização e iluminação.
O sistema inicial foi projetado para coletar dados de temperatura, luminosidade, consumo de energia, preparando o terreno para uma automação mais avançada e inteligente no futuro.
Funcionalidades Atuais:
Coleta de dados através de sensores de temperatura,luminosidade, consumo de energia.
Estrutura preparada para expansão com controle automatizado.
O ACLIoT está evoluindo para se tornar uma solução completa de automação, com foco em otimizar o uso de energia e melhorar o conforto dos usuários.
